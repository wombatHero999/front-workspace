동적 요소 생성

* 삽입 관련 메서드 part 1

선택된 요소 기준으로 새로운 요소를 추가하는 메서드

1. $(A).append(B) : A요소 내에 마지막에 B를 추가(자식으로써)
2. $(A).prepend(B) : A요소 내에 앞쪽에 B를 추가(자식으로써)
3. $(A).before(B) : A요소 바로 앞에(before) B를 추가(동위요소)
4. $(A).after(B) : A요소 바로 뒤에(after) B를 추가(동위)

* 삽인 관련 메서드 part2

1. $(B).appendTo(A) : B요소를 A요소 내에 마지막에 추가(자식)
2. $(B).prependTo(A) : B요소를 A요소 내에 앞에 추가(자식)
3. $(B).insertBefore(A) : B를 A 앞에 추가(동위)
4. $(B).insertAfter(A) : B를 A 뒤에 추가(동위)
