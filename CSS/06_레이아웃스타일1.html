화면 배치 방법 : display

        요소가 화면에 어떻게 보여질지(display)를 지정하는 속성
        - block : 화면을 수직 분할 (행으로 나눔)
        - inline : 화면을 수평 분할 (열로 나눔)
        - inline-block : inline의 수평분할 + block의 특징 결합(크기지정)
        - none : 화면에 요소를 보여주지 않도록 하는 속성
