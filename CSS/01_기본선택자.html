CSS(Cascading Style Sheets)

        HTML이 실제로 화면에
        표시되는 방법들을 기술하는 언어. 
        즉, HTML코드가 화면에 어떤 스타일로 보여질지를 정하는 언어이며, CSS의 몇 가지
        속성은 상위요소에 작성시 하위요소까지 연속적으로 전파(Cascading)된다.

CSS 기본 선택자

CSS 선택자의 종류

1. 전체 선택자 (*)

모든 요소들을 다 선택하고자 할 때 사용하는 선택자

2. 태그 선택자

html내부의 같은 태그들을 모두 선택하고자 할 때 사용하는 선택자

3. 클래스 선택자 (.클래스명)

현재 html문서 내에서 원하는 여러개의 요소를 선택하고자 할 때 사용

  • 클래스선택자1
  • 클래스선택자2
  • 클래스선택자3
  • 클래스선택자4
  • 클래스선택자5

4. 아이디 선택자(#아이디명)

현재 html문서 내에 특정 html요소 하나만을 선택 하고자 할 때 사용하는 선택자

  1. 아이디 선택자
  2. 아이디 선택자
  3. 아이디 선택자
  4. 아이디 선택자
