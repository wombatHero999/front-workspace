배치 관련 스타일

        position
         - 요소의 위치를 지정하는 속성
        
        position : relative;
         - position : relative속성이 지정된 요소가 하위 요소의 "상대적인" 기준이 되게하는 속성
         - position : relative가 부여된 요소의 하위 요소에는 top,bottom,left,right속성을 사용 할 수 있다.

        position : absolute; 
         - 기존 요소들의 배치 순서를 무시하고, 지정된 위치에 요소를 배치하는 속성. 
        
        position : fixed;
         - 요소를 항상 고정된 위치에 배치하는 속성
