FlexBox

        요소들의 정렬되는 방향, 순서, 간격등을 수치적으로 제시하지 않아도, 알아서 유연하게(flex)
        하게 처리해주는 스타일

        * Flexbox이용시 반드시 알아야 하는 개념

        1. Flexbox의 구성요소
         - flex-container : 정렬이 필요한 요소를 감싸는 부모 컨테이너
         - flex-item : 정렬이 적용되는 요소 

        2. Flexbox의 정렬 축
         - main axis 중심축
         - cross axis 교차축
