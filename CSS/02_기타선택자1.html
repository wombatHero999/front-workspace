1. 속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용하여 요소를 선택하는 선택자.
선택자 뒤에 []를 이용하여 속성과 속성값을 제시. 

        1. 기본선택자[속성=속성값]
        2. 기본선택자[속성~=속성값]
        3. 기본선택자[속성|=속성값]
        4. 기본선택자[속성^=속성값]
        5. 기본선택자[속성$=속성값]
        6. 기본선택자[속성*=속성값]

2. 자손/후손 선택자

자손 : 상위요소의 바로 아래에 존재하는 요소 후손 : 상위요소의 아래에 존재하는 모든 요소 

        *자손선택자
        a>b{속성 : 속성값;}

        *후손선택자
        a b{속성:속성값;}

3. 동위레벨 선택자

동위 관계에 있는 특정 요소를 선택할 때 사용하는 선택자 

        * a 요소 뒤에 있는 b 요소 "하나"를 선택하는 선택자
        a+b{속성:속성값;}

        * a 요소 뒤에 있는 b 요소 "전체"를 선택하는 선택자
        a~b{속성:속성값;}

4. 반응 선택자

사용자의 움직임에 따라 요소를 선택하는 선택자. 

        * 요소가 클릭이 되었을 경우의 요소를 선택하는 선택자
        선택자:active{속성:속성값;}

        * 요소에 마우스가 올라가는 요소를 선택하는 선택자
        선택자:hover{속성:속성값;}

5. 상태선택자

요소의 상태에 따라서 선택되는 선택자 

        1) checked(체크된) 상태의 요소를 선택하는 선택자
    
        2) 초점(focus)이 맞춰진 input요소만 선택하는 선택자 
        선택자:focus{속성:속성값;}

        3) 활성화된 요소를 선택하는 선택자

        4) 비활성화된 요소를 선택하는 선택자
        선택자:disabled
